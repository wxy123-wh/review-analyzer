## ADDED Requirements

### Requirement: 全界面中文化
系统 MUST 将前端用户可见文案统一为简体中文，且术语表达面向项目经理阅读场景。

#### Scenario: 首屏与导航文案中文一致
- **WHEN** 用户打开系统并完成登录
- **THEN** 页面标题、导航标签、状态标签、按钮文本均为简体中文，不出现英文模块标题

#### Scenario: 状态字段中文映射
- **WHEN** 前端渲染后端状态字段（如服务状态、占位状态、动作状态）
- **THEN** 用户可见区域展示为中文语义映射，不直接暴露原始英文枚举

### Requirement: 文案禁用词治理
系统 MUST 在项目经理视角页面中移除与业务价值无关或误导性的展示词汇。

#### Scenario: 炫技类词汇不可见
- **WHEN** 用户浏览登录页与主仪表盘文案
- **THEN** 不出现“炫技、PLACEHOLDER、测试模式、Quantum Console、Operator”等词

#### Scenario: 演示术语统一
- **WHEN** 页面需要描述非生产数据来源
- **THEN** 统一使用“演示数据”术语，不使用“测试数据/测试模式”术语
